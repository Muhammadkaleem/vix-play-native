# Feature Specs

Cross-cutting features that span multiple screens. Each defines capability, technical approach, edge cases, and acceptance criteria.

- `playback-engine.md` — Media3 + FFmpeg, codec/container matrix, HW/SW switching
- `subtitles.md` — formats, libass, online search, styling
- `gestures.md` — the gesture system (links to shipped code)
- `theming.md` — token system, Material You, skins, ambient backglow
- `background-pip.md` — background audio, media session, PiP
- `storage-scanning.md` — scoped storage, media scan, thumbnails
- `cloud.md` — cloud provider integration
- `monetization.md` — free tier, Pro, billing
